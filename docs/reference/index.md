# Package index

## All functions

- [`checkContextVars()`](https://weirichs.github.io/eatModel/reference/checkContextVars.md)
  : check for consistency of context variables and items

- [`checkLinking()`](https://weirichs.github.io/eatModel/reference/checkLinking.md)
  : check for linked design

- [`compareDefineModelObjects()`](https://weirichs.github.io/eatModel/reference/compareDefineModelObjects.md)
  :

  Compares two
  [`defineModel`](https://rdrr.io/pkg/eatModel/man/defineModel.html)
  objects

- [`correlationFromRes()`](https://weirichs.github.io/eatModel/reference/correlationFromRes.md)
  : Gives correlation estimates from the object returned by getResults()

- [`defineModel()`](https://weirichs.github.io/eatModel/reference/defineModel.md)
  : Prepares IRT analysis for Conquest, TAM, or mirt

- [`des2`](https://weirichs.github.io/eatModel/reference/des2.md) :
  Exemplary test design which may be typical in the context of
  large-scale assessment.

- [`des3`](https://weirichs.github.io/eatModel/reference/des3.md) :
  Exemplary test design for math and sciences (Biology, Chemistry,
  Physics).

- [`eapFromRes()`](https://weirichs.github.io/eatModel/reference/eapFromRes.md)
  : Collects all EAP results from the object returned by getResults()

- [`eapRelFromRes()`](https://weirichs.github.io/eatModel/reference/eapRelFromRes.md)
  : Gives EAP reliability from the object returned by getResults()

- [`eatModel-package`](https://weirichs.github.io/eatModel/reference/eatModel-package.md)
  : Specify and run IRT models from R (using Conquest, TAM, or mirt),
  including equating methods and parameter transformation.

- [`equat1pl()`](https://weirichs.github.io/eatModel/reference/equat1pl.md)
  : 1pl equating with optional elimination of linking DIF items

- [`get.dsc()`](https://weirichs.github.io/eatModel/reference/get.dsc.md)
  : Read Conquest ‘descriptives’ output files.

- [`get.equ()`](https://weirichs.github.io/eatModel/reference/get.equ.md)
  : Reads equivalence table created in Conquest analysis.

- [`get.itn()`](https://weirichs.github.io/eatModel/reference/get.itn.md)
  : Read ConQuest ‘itanal’ Output Files

- [`get.plausible()`](https://weirichs.github.io/eatModel/reference/get.plausible.md)
  : Read Conquest Plausible Values Output Files

- [`get.prm()`](https://weirichs.github.io/eatModel/reference/get.prm.md)
  : Read Conquest prm (parameter) Output Files

- [`get.shw()`](https://weirichs.github.io/eatModel/reference/get.shw.md)
  : Read ConQuest showfiles

- [`get.wle()`](https://weirichs.github.io/eatModel/reference/get.wle.md)
  : Read Conquest WLE or MLE Output Files

- [`getResults()`](https://weirichs.github.io/eatModel/reference/getResults.md)
  : Collect all results from Conquest, TAM, or mirt analyses into a
  common data frame

- [`itemFromRes()`](https://weirichs.github.io/eatModel/reference/itemFromRes.md)
  : Collect all item results from the object returned by getResults()

- [`multiEquatError()`](https://weirichs.github.io/eatModel/reference/multiEquatError.md)
  : 1pl linking errors for three measurement occasions

- [`plotDevianceConquest()`](https://weirichs.github.io/eatModel/reference/plotDevianceConquest.md)
  :

  Deviance Plot for Conquest log files

- [`plotDevianceMirt()`](https://weirichs.github.io/eatModel/reference/plotDevianceMirt.md)
  :

  Deviance Plot for mirt objects

- [`plotICC()`](https://weirichs.github.io/eatModel/reference/plotICC.md)
  : Plots item characteristic curves.

- [`pvFromRes()`](https://weirichs.github.io/eatModel/reference/pvFromRes.md)
  : Collects all plausible values from the object returned by
  getResults()

- [`q3FromRes()`](https://weirichs.github.io/eatModel/reference/q3FromRes.md)
  : Extracts the Q3 matrices from the object returned by getResults()

- [`reading`](https://weirichs.github.io/eatModel/reference/reading.md)
  : Dichotomous and partial credit responses for reading comprehension
  data obtained from large-scale assessment

- [`regcoefFromRes()`](https://weirichs.github.io/eatModel/reference/regcoefFromRes.md)
  : Gives regression coefficients (betas) from the object returned by
  getResults()

- [`replaceLinkingError()`](https://weirichs.github.io/eatModel/reference/replaceLinkingError.md)
  :

  Output interface for
  [`multiEquatError`](https://rdrr.io/pkg/eatModel/man/multiEquatError.html)

- [`runModel()`](https://weirichs.github.io/eatModel/reference/runModel.md)
  : Run IRT model specified by 'defineModel' using Conquest, TAM, ort
  mirt

- [`simEquiTable()`](https://weirichs.github.io/eatModel/reference/simEquiTable.md)
  : Computes equivalence table for the Rasch and partial credit model

- [`splitModels()`](https://weirichs.github.io/eatModel/reference/splitModels.md)
  : Create several models by splitting the qMatrix and/or person.groups

- [`transformToBista()`](https://weirichs.github.io/eatModel/reference/transformToBista.md)
  : Transformation of item and person parameters to the Bista metric.

- [`trends`](https://weirichs.github.io/eatModel/reference/trends.md) :
  Reading and listening achievement test data obtained from large-scale
  assessment at three times of measurement

- [`wleFromRes()`](https://weirichs.github.io/eatModel/reference/wleFromRes.md)
  : Collects all WLEs from the object returned by getResults()

- [`wleRelFromRes()`](https://weirichs.github.io/eatModel/reference/wleRelFromRes.md)
  : Gives WLE reliability from the object returned by getResults()
